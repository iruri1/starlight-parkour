# 星际暖光跑酷

## 项目介绍
一款可爱的科幻跑酷游戏，玩家控制角色在星际场景中奔跑、跳跃、躲避障碍。

## 技术栈
- 前端：Cocos Creator 3.8 + TypeScript
- 后端：Node.js + Express + MongoDB
- 工具：VS Code + Git + GitHub

## 游戏特色
- 🚀 科幻未来风格，可爱炫酷的视觉体验
- 🎮 简单易上手，适合所有年龄段玩家
- 🌟 丰富的角色和道具系统
- 👥 社交功能，与好友一起挑战
- 🏆 成就系统，收集和挑战乐趣无穷

## 核心玩法
- 上下滑动手势：跳跃和滑行
- 左右滑动手势：切换跑道
- 长按：短暂加速
- 双击：特殊技能瞬移
- 收集能量块，躲避障碍物
- 挑战BOSS，成为星际跑酷王者

## 角色介绍
- ⭐ **星尘猎手**：勇敢的小机器人，擅长加速冲刺
- 🌌 **时空信使**：神秘的外星访客，掌握时空技能
- 🚀 **天际行者**：可靠的太空探险家，跳跃能力出众

## 道具系统
- 🛡️ **能量护盾**：抵御碰撞伤害
- ⏰ **时光加速器**：减慢障碍物速度
- 🌊 **引力波发射器**：吸附周围能量块
- 🧲 **磁力核心**：自动收集能量块
- ⚡ **速度增幅芯片**：提升移动速度
- 💊 **应急修复剂**：自动恢复护盾

## 快速开始
```bash
# 克隆项目
git clone https://github.com/你的用户名/starlight-parkour.git

# 进入项目目录
cd starlight-parkour

# 启动前端开发
cd client
# 使用Cocos Creator打开项目

# 启动后端服务
cd server
npm install
npm run dev
```
